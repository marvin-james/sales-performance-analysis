# Sales Performance & Customer Insights Analysis

## Project Overview
This project analyzes sales transaction data to identify revenue trends and top-performing products.

## Dataset
Sales transaction data stored in CSV format.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Electronics products generate the highest revenue
- Sales vary by region
- Monthly revenue trends show business growth

## How to Use
This project is for portfolio demonstration and analysis purposes.
